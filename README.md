# 🎯 Cloning the project

```bash
  git clone https://github.com/isabella-projects/blossoming-flowers.git
```

## 📌 Go to the project directory

```bash
  cd blossoming-flowers
```

# 📐 Setup Instructions

1. Install Node Dependencies: In the project directory run:

```bash
  npm install
```

2. Start the Vite Development Server:

```bash
  npm run dev
```
